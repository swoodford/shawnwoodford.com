Pure CSS Terminal and Line Numbering

CSS Styles for a terminal emulator (ie, the window you have in Linux or OSX that presents a UNIX command line interface). It can also do line numbering in a way that looks like a popular vim theme. It does this all through advanced :after and :before trickery!

Try <pre class="terminal-emulator"> or <div class="terminal-emulator"> to use. For the line numbering, add the vim class, like <div class="terminal-emulator vim">

Vaguely modeled off of GNOME 3's styling. Should work with or without Bootstrap.
